RULES

In Minesweeper, the goal is to clear all non-mine cells by clicking them, using numbers on revealed cells to indicate adjacent mines.

  • 1. Begin by left-clicking any cell on the board.
  • 2. Each number on a revealed cell shows how many mines are in the eight surrounding cells (including diagonals), while a blank cell means no adjacent mines and automatically uncovers all connected blank cells.

  • 3. Use the right-click button to place a flag on a cell where you believe a mine is located. This prevents you from accidentally clicking it.
  • 4. Use the numbers as clues to determine which other cells are safe to click.
  • 5. The game is won when all cells that are not mines have been revealed.

HOURS PLAYED

Minesweeper originated from earlier puzzle games, most notably Ian Andrew's Mined-Out (1983), before being developed by Robert Donner and Curt Johnson for the Microsoft Entertainment Pack in 1990. The game was included in Windows 3.1 and was designed to help users familiarize themselves with the Windows graphical user interface (GUI) and mouse controls. While the original version was eventually replaced in newer Windows versions, its iconic gameplay has made it a classic logic puzzle known worldwide.
